HERA PWA Removal

Multi-tenant Safety Update

Important: HERA has removed PWA functionality to ensure proper multi-tenant data isolation and security. This page helps clean up any existing PWA installations.

Manual Removal Instructions:

  1. Chrome: Go to chrome://settings/content/all → Find HERA → Click delete
  2. Firefox: about:preferences#privacy → Clear Site Data
  3. Safari: Preferences → Privacy → Manage Website Data → Remove HERA
  4. Edge: edge://settings/content/all → Find HERA → Remove

Mobile Devices:

  1. iOS: Settings → Safari → Advanced → Website Data → Remove HERA
  2. Android: Chrome Settings → Site Settings → All Sites → HERA → Clear & Reset
