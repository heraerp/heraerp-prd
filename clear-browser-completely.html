Manual Steps (Do this first!):

  1. Stop the dev server: Press Ctrl+C in terminal

    • Find localhost:3000
    • Click it and then click Delete data
  3. Clear Next.js cache:
    rm -rf .next
  4. Restart the dev server:
    npm run dev
